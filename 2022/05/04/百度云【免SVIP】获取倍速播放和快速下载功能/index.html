百度云【免SVIP】获取倍速播放和快速下载功能

发表于 2022-05-04

一、获取【百度云免SVIP 】倍速播放功能

所需原料：

  • 微软自带的Edge浏览器
  • 油猴插件
  • 倍速播放脚本

1.1 安装油猴插件

  1. 油猴插件地址： Tampermonkey - Microsoft Edge Addons 在Edge打开后，点击获取即可，如下图所示：
  1. 安装完成后，在Edge图示位置依次点击【1】、【2】，查看扩展的插件是否安装成功。

【注】：到这里油猴插件的安装就大功告成了。

1.2 安装倍速播放脚本

  1. 倍速播放脚本地址 点击即可： HTML5视频播放工具 (greasyfork.org) ，打开后，安装此脚本：

  1. 依次点击图示箭头位置，查看安装的倍速播放脚本是否安装成功，如下图所示：

  1. 测试脚本运行情况，打开百度网盘的视频，按键盘的【C键】，每按一次速度加0.1，如图所示，左上角播放速度2.3倍，竟然比百度的2.0倍速还快… … 简直是一个神器！这个倍速度有没有上限就不知道了，可自行测试。按键盘的【X键】，减速播放。按【Z键】，恢复正常播放速度。

【注】：更多快捷键，可点击油猴插件–>快捷键，查看。也可直接访问：倍速播放插件-快捷键

二、在微软Edge浏览器获取【百度云免SVIP】 快速下载功能

要获取这个功能，需要准备三个文件：

  • IDM Windows 桌面版

  • IDM浏览器集成插件

  • 网盘直链下载助手

2.1 下载并安装IDM到电脑

下载地址： IDM下载器中文破解版_IDM下载器中文破解版下载-下载之家 (downza.cn) 打开网页，下拉找图示位置，下载好后，傻瓜式安装。

【注】：因为这个IDM只有30天免费试用期限，所以找了这个破解版。

2.2 下载IDM浏览器集成扩展插件

地址： IDM Integration Module - Microsoft Edge Addons

【注】：这两个IDM缺一不可，两个IDM里应外合，就白嫖的百度云盘文件里的下载直链。
